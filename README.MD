## 📖 Overview

This repository hosts a strictly maintained `.m3u` playlist for Japanese IPTV streaming. It provides direct streaming links to publicly available, free-to-air (FTA) Japanese broadcast networks and internet channels. 

Whether you want to practice your Japanese, catch up on Tokyo news, or stream live anime broadcasts, this playlist is formatted for immediate use in any modern media player.

## 📺 Channel Categories

The playlist is organized into the following groups for easy navigation:

* **News & Weather:** 24/7 live news broadcasts from major networks (e.g., NHK, TBS, NTV).
* **Anime & Kids:** Dedicated animation channels and family programming.
* **Entertainment & Variety:** Talk shows, J-Pop music channels, and local broadcast streams.
* **Sports:** Live regional and national sporting events.

## 🚀 How to Use

You can load this playlist directly into most modern media players and IPTV clients. 

### Recommended Clients
* **Windows / macOS / Linux:** [VLC Media Player](https://www.videolan.org/), mpv
* **Android / Android TV:** TiviMate, OTT Navigator
* **iOS / tvOS (Apple TV):** iPlayTV, GSE Smart IPTV

### Setup Instructions (VLC Media Player)
1. Download and open VLC.
2. Go to **Media** > **Open Network Stream...** (or press `Ctrl + N`).
3. Paste the raw URL of the playlist:
   ```text
   https://raw.githubusercontent.com/obbiiall1/japan-iptv-playlist/refs/heads/main/jp.m3u
   ```
